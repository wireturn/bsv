## paymail whois

Find a paymail handle across several providers

### Synopsis

```
        .__           .__        
__  _  _|  |__   ____ |__| ______
\ \/ \/ /  |  \ /  _ \|  |/  ___/
 \     /|   Y  (  <_> )  |\___ \ 
  \/\_/ |___|  /\____/|__/____  >
             \/               \/
```


Search 8 public paymail providers for a handle.

```
paymail whois [flags]
```

### Examples

```
paymail whois mrz
paymail w mrz
paymail w \$mr-z
paymail w 1mrz
```
